Project Title: Superstore Sales Performance Dashboard
📊 Project Overview
This project features an interactive Excel dashboard designed to analyze retail performance data from 2014 to 2017. By transforming raw transactional data into visual insights, the dashboard tracks profitability, sales growth, and regional performance to support data-driven business decisions.

🛠️ Technical Features
Data Analysis: Utilized Pivot Tables and Advanced Formulas to calculate metrics like a Grand Total Profit of 286,397 and a Total Sales volume of 2.29M.

Interactive UI: Implemented Slicers (Segment, Category, Region) and a Timeline (Order Date) to allow users to filter data dynamically.

Data Visualization: Created a cohesive visual layout featuring:

Line Charts for trend analysis.

Donut and Pie Charts for categorical and regional distribution.

Bar Charts for ranking top-performing products.

📈 Key Insights from the Data
Steady Growth Trend: Sales have shown a consistent upward trajectory from 2014, peaking in 2017 at over 483,000.

Regional Dominance: The West region is the strongest performer, accounting for 32% of total sales, followed closely by the East.

Customer Segmentation: The Consumer segment is the primary driver of volume, generating nearly 800,000 in sales—significantly higher than the Corporate or Home Office segments.

Product Performance: The Canon imageCLASS 2200 Advanced Copier stands out as the highest-selling individual product.

Profitability: Maintained an average profit margin of approximately 12% across all categories.
